## Overview
This project is a full-stack web application that leverages Google Gemini AI to provide:
- Personalized job recommendations
- Real-time career mentorship via chatbot
- Actionable career tips and market trends by sector

Built with a **React** frontend and a **FastAPI** backend, it is designed for job seekers, students, and professionals looking to make informed career decisions.

---

## Features
- **AI-Powered Job Recommendations:**
  - Enter your preferences, skills, and detailed expectations to get tailored job matches.
  - Each job includes title, company, description, skills, and a relevance explanation.
- **AI Chat Mentor:**
  - Chat with an AI for career advice, interview tips, or job search strategies.
  - Markdown support for rich, readable responses.
- **Career Tips & Market Trends:**
  - Select a sector (e.g., Technology, Finance, Healthcare) to get actionable tips and trends.
  - Tips are dynamically generated by the AI for each sector.
- **Modern UI/UX:**
  - Responsive, accessible, and visually appealing interface.
  - Consistent design system with custom CSS.
- **Environment Safety:**
  - Sensitive data (API keys, .env) is protected via `.gitignore`.

## Setup & Installation

### Backend (FastAPI)
1. **Navigate to backend:**
   ```sh
   cd backend
   ```
2. **Install dependencies:**
   ```sh
   pip install -r requirements.txt
   ```
3. **Set up environment:**
   - Create a `.env` file in the root or backend folder:
     ```
     GEMINI_API_KEY=your_gemini_api_key_here
     ```
4. **Run the server:**
   ```sh
   uvicorn main:app --reload
   ```
   - The API will be available at `http://localhost:8000`

### Frontend (React)
1. **Navigate to frontend:**
   ```sh
   cd frontend
   ```
2. **Install dependencies:**
   ```sh
   npm install
   ```
3. **Start the development server:**
   ```sh
   npm start
   ```
   - The app will run at `http://localhost:3000`

---

## Environment Variables
- **Gemini API Key:** Required for backend AI features. Set `GEMINI_API_KEY` in your `.env` file.
- **Frontend API URL:** Optionally set `REACT_APP_API_URL` in `frontend/.env` to override the backend URL (default: `http://localhost:8000`).

## Usage Guide

### Job Recommendations
- Go to the **Recommendations** page.
- Enter your general preferences (e.g., "Remote, Full-time, San Francisco"), key skills, and any detailed expectations (salary, company culture, etc.).
- Click **Get Personalized Recommendations**.
- View a list of AI-matched jobs, each with skills, relevance notes, and a link to the full posting.

### AI Chat Mentor
- Go to the **Chat** page.
- Ask any career-related question (e.g., "How do I prepare for a React interview?", "What skills are in demand for data science?").
- The AI mentor responds in real time, supporting markdown formatting for lists, links, and emphasis.

### Career Tips & Market Trends
- Go to the **Career Tips** page.
- Select a sector (e.g., Technology, Finance, Healthcare).
- Instantly receive 5 actionable tips and trends for your chosen sector, generated by the AI.

## API Endpoints

### POST `/recommendations`
- **Request:**
  ```json
  {
    "preferences": "Remote, Full-time, San Francisco",
    "skills": "React, Node.js, Python",
    "detailed_expectations": "Looking for a collaborative team, salary $90k+"
  }
  ```
- **Response:**
  ```json
  {
    "recommendations": [
      {
        "title": "Frontend Developer",
        "company": "TechCorp",
        "url": "https://...",
        "short_description": "...",
        "skills": ["React", "JavaScript"],
        "relevance_notes": "Matches your skills and remote preference."
      },
      ...
    ]
  }
  ```

### POST `/chat`
- **Request:** `{ "message": "How do I improve my resume?" }`
- **Response:** `{ "response": "...AI-generated answer..." }`

### GET `/market-trends?sector=Technology`
- **Response:**
  ```json
  {
    "market_trends": [
      { "title": "Emphasize Cloud Skills", "description": "Cloud computing is in high demand..." },
      ...
    ]
  }
  ```

- **Deployment:**
  - Use services like Vercel, Netlify (frontend) and Render, Heroku (backend) for production.
  - Set environment variables securely in your deployment platform.
